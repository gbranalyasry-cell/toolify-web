# 🌐 Toolify

A multi-tool web platform with a modern dark interface that offers:
- 📄 PDF to Word conversion
- 🖼️ Image compression
- 🔗 QR Code generation

## 🚀 Features
- Stylish dark mode design, mobile-friendly
- Arabic & English support
- Built with Flask + HTML + CSS
